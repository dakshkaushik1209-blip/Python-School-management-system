# School Management System

A simple School Management System built using Python and Object-Oriented Programming (OOP) concepts.

## Features

### Student Management

* Register new students
* Prevent duplicate roll numbers
* Store student details in JSON database
* Add subject-wise grades
* View complete student details
* Calculate percentage automatically

### Teacher Management

* Register new teachers
* Prevent duplicate employee IDs
* View teacher details
* Store teacher records in JSON database

### Validation

* Email validation
* Duplicate record checking

## Concepts Used

* Object-Oriented Programming (OOP)
* Abstraction using Abstract Base Classes (ABC)
* Inheritance
* Polymorphism
* Static Methods
* File Handling
* JSON Data Storage
* Dictionaries and Lists

## How to Run

1. Clone the repository

```bash
git clone https://github.com/dakshkaushik1209-blip/Python-School-management-system.git
```

2. Open the project folder

```bash
cd School-Management-System
```

3. Run the program

```bash
python "School manage system.py"
```
